PlutoPay Case Study

Main focus:

Create an application from scratch. Grow user experience skills.

Skills utilized:

Figma, user personas, user journey, information architecture, usability testing, and prototyping

Introduction

Problem:

With the boom of the ecommerce market, the problem to make purchases securely and efficiently was brought forward. How can users track all of their finances and make money transfer safely in one tool?

Solution:

Create a responsive web app that potential users can use to transfer money, create goals and budgets, and track all of their money without the need to visit a physical bank or store.

This project was reviewed with CareerFoundry's UX Program.
